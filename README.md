## Project Overview
-  소고기 단면을 이용한 품질 평가
  - 실생활에서 내가 먹는 고기의 육질을 평가해주는 한우 등심 분류기

-  프로젝트 배경 및 기대효과
  - 소비자로서 실생활에서 내가 먹는 소고기 등급이 맞는 지 검증
  - 부스트캠프를 통해서 배운 AI 지식을 실생활에 활용
